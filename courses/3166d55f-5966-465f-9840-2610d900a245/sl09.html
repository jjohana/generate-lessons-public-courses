MRU et MRUA : deux modèles rectilignes de base

Identifier les hypothèses, les grandeurs constantes et les lois horaires à appliquer.

Type

Slide de cours

Synthese visuelle

Comparer les deux modèles

Critère MRU MRUA
Trajectoire Rectiligne ; une seule coordonnée suffit Rectiligne ; une seule coordonnée suffit
Grandeur constante Vitesse constante Accélération constante
Accélération Nulle : a = 0 Constante, en général non nulle
Loi de vitesse v(t) = v v(t) = v₀ + at
Loi de position x(t) = x₀ + vt x(t) = x₀ + v₀t + 1/2 at²
Usage Déplacement sans variation de vitesse Variation régulière de vitesse sur un intervalle étudié

Hypothèses et méthode de reconnaissance

  • Le modèle est valable si le mouvement peut être décrit dans un référentiel choisi par une seule coordonnée x(t) ; dès que la trajectoire n’est plus droite, ces lois ne s’appliquent plus telles quelles.
  • Reconnaître le cas étudié revient à chercher la grandeur constante : vitesse constante pour le MRU, accélération constante pour le MRUA ; une accélération variable impose un autre modèle.
  • Méthode pratique : fixer x₀ et v₀ à l’instant initial, vérifier les unités (m, m·s⁻¹, m·s⁻²), puis choisir la loi qui répond directement à la question posée.
  • Exemples typiques : palier de vitesse d’un mobile en translation pour le MRU ; phase de départ, freinage régulier ou montée en régime d’un organe mécanique pour le MRUA.

Lois horaires à retenir

\[\text{MRU : } a=0,\quad x(t)=x_0+vt\\ \text{MRUA : } a=\text{cste},\quad v(t)=v_0+at,\quad x(t)=x_0+v_0t+\frac12at^2\]
